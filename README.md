### Installation

```bash
yarn add react-flag-kit
```

### Usage

```js
import { FlagIcon } from "react-flag-kit";

export const USAFlag = () => <FlagIcon code="US" size={48} />;
```

More information you can find in [docs](https://umidbekk.github.io/react-flag-kit).
