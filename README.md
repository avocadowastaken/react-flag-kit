### Installation

```bash
yarn add react-flag-kit
```

### Usage

```js
import { FlagIcon } from "react-flag-kit";

export default () => <FlagIcon code="WW" size={48} />;
```

More information you can find in [docs](https://umidbekkarimov.github.io/react-flag-kit).
